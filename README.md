# Telecom-X
Desafio 2 Alura: Telecom X

## 📡 Telecom X - Estratégia de Retenção e Análise de Churn ##

### 📝 Sobre o Projeto ###
Este projeto foi desenvolvido para enfrentar um dos maiores desafios do setor de telecomunicações: a evasão de clientes (Churn). Através de uma análise profunda nos dados da Telecom X, o objetivo foi identificar os "gargalos" que fazem os clientes cancelarem seus serviços e propor soluções baseadas em dados.O diferencial desta análise é a união da Ciência de Dados com a Visão Estratégica de Marketing, focando não apenas nos números, mas na experiência e fidelização do usuário.

### 🛠️ Tecnologias e Ferramentas ### 
Python: Linguagem principal para manipulação lógica.Pandas: Extração e tratamento de dados (ETL).Seaborn & Matplotlib: Visualizações estatísticas e identificação de padrões.Google Colab: Ambiente de desenvolvimento em nuvem.

### 🚀 Estrutura do Notebook ### 
O projeto seguiu uma trilha lógica de análise profissional:📌 Extração: Captura de dados brutos via API/JSON.🔧 Transformação: Normalização de dados aninhados e tratamento de inconsistências (limpeza de nulos e conversão de tipos).📊 Carga e Análise (EDA): Criação de visualizações para descobrir os vilões da evasão.

### 📄 Relatório Final: ###
Insights de negócio e recomendações estratégicas.💡 Principais InsightsTaxa de Churn: Identificamos uma perda de $26,5\%$ da base de clientes.O Vilão Contratual: Clientes com contrato Mês a Mês possuem uma taxa de evasão de $42,7\%$, evidenciando a fragilidade desse modelo de venda.O Vale da Morte: A maioria das desistências ocorre nos primeiros 10 meses de contrato, sugerindo uma falha na retenção de novos clientes.Paradoxo do Preço: Clientes de Fibra Óptica pagam mensalidades mais altas, porém apresentam maior tendência de cancelamento.🏃
